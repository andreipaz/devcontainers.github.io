Funcionalidades Disponíveis

Essa tabela contém todas as funcionalidades de Dev Container oficiais e as desenvolvidas pela comunidade, e conhecidas até o momento do crawling das collections registradas. Essa lista é continuamente atualizada com as informações das últimas funcionalidades. Dê uma olhada no repositório de feature quick start para adicionar a sua própria!

Para referenciar uma feature adicione à seção `features` de um devcontainer.json.

Se precisar relatar algo sobre uma Feature, faça isso no registry onde a Feature está hospedada.

Para adicionar sua própria coleção à lista, crie um PR editando esse arquivo yaml.
